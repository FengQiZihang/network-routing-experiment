# 数据通信与网络技术 - 实验操作练习

本仓库用于记录《数据通信与网络技术》（交换路由技术）课程的实验操作练习，包含 VLAN 划分、STP 配置、链路聚合、OSPF 路由、ACL 访问控制等网络技术实验。所有配置文件和实验步骤均为课堂学习阶段完成，仅供教学与个人学习参考。

## 仓库结构

```
├── 个人实验操作练习/              # 课堂同步练习 (10个实验)
├── 教学平台布置8次实验/           # 教学平台作业 (6个实验已完成)
├── 2022级交换路由期末考试题/      # 期末考试配置题
├── eNSP软件环境搭建/              # 环境搭建指南和工具
├── .gitignore
└── README.md
```

## 实验环境

- **eNSP** (Enterprise Network Simulation Platform) v1.3.00+
- **华为路由器/交换机设备模拟**
- **Windows 11 操作系统**

## 使用方法

1. **环境准备**：参考 `eNSP软件环境搭建/` 目录完成环境安装
2. **实验学习**：按照实验编号顺序进行学习，建议从 VLAN 配置开始
3. **文件说明**：
   - `.topo` - eNSP拓扑文件
   - `flash.efz` / `vrpcfg.zip` - 设备配置文件
   - `.md` / `.txt` - 实验说明文档
   - `.png` - 拓扑图
