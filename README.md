# 数据通信与网络技术 - 实验操作练习

📘 本仓库用于记录《数据通信与网络技术》课程的实验操作练习，包含VLAN划分、STP配置、链路聚合、OSPF路由、ACL访问控制等网络技术实验。所有配置文件和实验步骤均为课堂学习阶段完成，仅供教学与个人学习参考。

## 🎯 实验内容

- [x] **01_基于接口划分VLAN** - VLAN基础划分和配置
- [ ] **02_stp配置实验** - 生成树协议配置与优化
- [x] **03_链路聚合** - 以太网链路聚合技术
- [x] **04_OSPF实验** - OSPF基础路由配置
- [ ] **05_OSPF综合实验** - OSPF高级配置和应用
- [x] **06_OSPF第三次实验** - OSPF进阶练习
- [x] **07_Vlan间通信** - 跨VLAN通信实现
- [x] **08_ACL-基础** - 访问控制列表基础配置
- [ ] **09_ACL-综合** - ACL高级配置进行中
- [x] **10_3A认证** - 3A认证网络安全管理机制

## 🛠️ 实验环境

- **eNSP (Enterprise Network Simulation Platform)** v1.3.00+
- **华为路由器/交换机设备模拟**
- **Windows 11 操作系统**
